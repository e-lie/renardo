## Shortcut Overview

Below is a quick reference table of the most essential shortcuts:

| Action | Windows/Linux | Mac |
|--------|--------------|-----|
| Execute current line | Alt+Enter | Option+Enter |
| Execute paragraph/selection | Ctrl+Enter | Cmd+Enter |
| Stop all sound | Ctrl+. | Cmd+. |
| New tab | Ctrl+T | Cmd+T |
| Close tab | Ctrl+W | Cmd+W |
| Switch tab | Ctrl+Tab | Cmd+Tab |
| Toggle Zen Mode | F11 or Ctrl+Shift+Z | F11 or Cmd+Shift+Z |
| Save session | Ctrl+S | Cmd+S |
| Clear console | Ctrl+L | Cmd+L |
