# API Reference

This section provides technical information for developers who want to integrate with or extend Renardo.

## WebSocket API

Renardo uses WebSockets for real-time communication between the editor and the server. The WebSocket API enables:

- Code execution
- State updates
- Console messages
- Event notifications

For details, see the [WebSocket API documentation](./websocket.md).

## REST API

Renardo also provides a REST API for various operations:

- Managing sessions
- Accessing tutorial files
- Working with startup files
- Accessing documentation

See the [REST API documentation](./rest.md) for more information.

## Editor Integration

If you want to integrate Renardo with other tools or create your own client, see the [Editor Integration guide](./editor-integration.md).

## Extending Renardo

Renardo can be extended with custom:

- Instruments
- Effects
- Pattern types
- Player methods

See the [Extension API documentation](./extensions.md) for details on creating your own extensions.
